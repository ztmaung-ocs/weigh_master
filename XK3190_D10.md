Print Data
1. Press [format print], indicator display [Pb 00],
   . 80 [Input] → print calibration parameters
   . 81 [Input] → print communication parameters
   . 82 [Input] → print print parameters
   . 89 [Input] → print all parameters


2. Set the indicator’s serial settings

    On the XK3190-D10, the PC/RS232 settings are under parameter type 01:

    Press [parameter]

    Type 01, press [INPUT] → communication parameters menu

    Set these:

    Adr → device address (can just use 01)

    bt → baud rate:

    0: 600

    1: 1200

    2: 2400

    3: 4800

    (later in calib: 4: 9600)
    Pick something Node-RED supports easily, e.g. bt 3 = 4800 or bt 4 = 9600.

    tF → mode

    0 = continuous sending

    1 = on command

    For Node-RED, easiest is: Adr 01, bt 4 (9600), tF 0 (continuous).
